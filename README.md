# Hey, I’m Marc 👋

I build small, practical AI tools around retrieval, search, local LLMs, and production GenAI systems.

Most of my work is focused on making AI tooling lighter, faster, and less painful to actually use — especially for smaller models, local setups, and resource-constrained environments.

## Current focus

- **TinySearch** — lightweight web research for MCP agents
- **Query Autocomplete** — local typeahead/autocomplete for RAG and chat apps
- RAG systems, document intelligence, search pipelines, and local-first AI tooling

## Why I build

I like tools that solve boring but real problems:

- less context waste
- better retrieval quality
- faster local workflows
- lower API bills
- fewer overengineered “AI wrapper” layers

If something makes agents more useful without burning the whole context window, I’m probably interested.

## Projects

### TinySearch

TinySearch lets your LLM search the web without dumping entire pages into context.

It searches, crawls, reranks, and returns compact source-grounded chunks that are easier for small/local models to use.

### Query Autocomplete

A lightweight autocomplete/typeahead tool for apps that need local suggestions from user documents or text, without setting up a full search backend.
